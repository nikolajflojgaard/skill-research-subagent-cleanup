# Research Subagent Cleanup Skill

AgentSkill for using subagents only when they genuinely help and cleaning them up afterward.

## Use

Install or copy `SKILL.md` into your agent skills directory.

The skill is intentionally generic: adapt the spawn, wait, and close commands to your agent runtime.

## Principle

Subagents are for useful parallelism, independent investigation, and risk reduction. They are not decoration.
